## ✨ 核心功能

### 📊 仪表板
- 本月收入、支出、结余一览
- 账户总资产统计
- 最近交易记录

- 快速记录收入和支出

- 7种收入分类

- 完整的交易记录

- 清晰的交易列表展示

- 多账户支持
- 现金、银行卡、支付宝等
- 实时余额更新
- 账户总资产统计

### 🎯 预算管理
- 预算设置和提醒
- 支出控制

### 📈 财务报表
- 按月/季度/年/全部统计
- 收入支出对比
- 清晰的数据分析

### 💾 数据管理
- 导出为 JSON 文件
- 导入备份数据
- 清空所有数据

## 🆘 常见问题

### Q: 数据会丢失吗？
A: 不会。数据保存在浏览器本地存储中，除非：
- 清空浏览器数据
- 卸载浏览器
- 切换浏览器或设备

**建议定期导出数据备份**

### Q: 支持多设备同步吗？
A: 目前不支持云同步。如需在多设备使用：
1. 在设备A导出数据
2. 在设备B导入数据

### Q: 可以在微信小程序上用吗？
A: 目前是网页版本，可在微信中打开链接或扫描二维码使用。
如需真正的微信小程序，可查看"升级建议"章节。

### Q: 支持哪些浏览器？
A: 支持所有现代浏览器：
- Chrome 90+
- Safari 14+
- Firefox 88+
- Edge 90+

### Q: 有导入导出功能吗？
A: 有！在"设置"页面可以：
- 导出数据为 JSON 文件
- 导入之前备份的 JSON 文件
- 完整备份所有交易记录
