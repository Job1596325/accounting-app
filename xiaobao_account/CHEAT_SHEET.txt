第1步：注册 GitHub
  1. 访问 https://github.com
  2. 点击 Sign up
  3. 填写邮箱、密码、用户名
  4. 验证邮箱
  ✅ 完成

第2步：创建仓库
  1. 点击头像 → Your repositories
  2. 点击绿色 New
  3. 名称：accounting-app
  4. 选择 Public ✅
  5. 勾选 Add a README file ✅
  6. 创建
  ✅ 完成

第3步：上传文件
  1. 点击 Add file → Upload files
  2. 拖拽文件：
     - pro.html
     - index-standalone.html
     - qrcode-generator.html
  3. Commit changes
  ✅ 完成

第4步：启用 Pages
  1. Settings → Pages
  2. Branch: main
  3. Folder: / (root)
  4. Save
  5. 等待 1-5 分钟
  ✅ 完成

问题：打开是白页
解决：
  1. 检查 URL 是否正确
  2. 刷新浏览器（Ctrl+F5 强制刷新）
  3. 等待 5-10 分钟
  4. 检查 Pages 是否已启用

问题：找不到应用链接
解决：
  1. 进入仓库 Settings
  2. 选择 Pages
  3. 确认 Branch 为 main
  4. 确认 Folder 为 / (root)

问题：文件上传失败
解决：
  1. 检查网络连接
  2. 确认仓库是 Public
  3. 重新刷新页面
  4. 重新上传

问题：在微信中打不开
解决：
  1. 确保链接是 https:// 开头
  2. 检查链接是否包含 /pro.html
  3. 在浏览器中先测试链接
  4. 尝试用二维码扫码
