✅ 方式1：立即本地使用（最简单，无需任何配置）
   ────────────────────────────────────────────
   1. 找到 pro.html 文件
   2. 双击打开
   3. 立即开始记账！

   ⏱️  时间：0 分钟
   📍 可访问：仅本机
   💻 需求：浏览器


✅ 方式2：部署到 GitHub Pages（推荐，全网可用）
   ────────────────────────────────────────────
   1. 阅读 QUICK_START.md（5 分钟快速指南）
   2. 注册 GitHub 账号
   3. 创建仓库并上传文件
   4. 启用 GitHub Pages
   5. 获得应用链接，全网可用！

   ⏱️  时间：10 分钟
   📍 可访问：全球（任何设备、任何地点）
   💻 需求：GitHub 账号（免费）
   📱 微信：可在微信中打开或添加到主屏幕


✅ 方式3：使用自动化脚本（最快，适合开发者）
   ────────────────────────────────────────────
   Windows: 双击 upload-to-github.bat
   Mac/Linux: 运行 ./upload-to-github.sh

   ⏱️  时间：5 分钟
   📍 可访问：全网可用
   💻 需求：Git（自动脚本安装）
