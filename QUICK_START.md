# 🚀 GitHub 上传快速开始

## ⏱️ 5分钟快速教程

### 📍 第一步：注册 GitHub（2分钟）

1. **打开浏览器**
   - 访问：https://github.com

2. **点击 Sign up**
   ![点击注册](https://img.shields.io/badge/按照步骤-1-blue)

3. **填写信息**
   - Email：您的邮箱
   - Password：您的密码
   - Username：用户名（记住这个！）

4. **验证邮箱**
   - 检查邮箱，点击验证链接

✅ **完成！** GitHub 账号已创建

---

### 📍 第二步：创建仓库（1分钟）

1. **登录后，点击右上角头像**

2. **选择 "Your repositories"**

3. **点击绿色 "New" 按钮**

4. **填写信息：**
   ```
   Repository name: accounting-app
   Description: 小宝记账 - 专业级记账理财应用
   Public: ✅ 勾选（必须！）
   Add a README file: ✅ 勾选
   ```

5. **点击 "Create repository"**

✅ **完成！** 仓库已创建

---

### 📍 第三步：上传文件（1分钟）

1. **在仓库页面，点击 "Add file"**

2. **选择 "Upload files"**

3. **拖拽文件到上传框**

   需要上传这些文件：
   - ✅ `pro.html`（必须）
   - ✅ `index-standalone.html`（推荐）
   - ✅ `qrcode-generator.html`（推荐）

4. **点击 "Commit changes"**

✅ **完成！** 文件已上传

---

### 📍 第四步：启用 Pages（1分钟）

1. **进入仓库页面**

2. **点击 "Settings"（设置）**

3. **左边选择 "Pages"**

4. **在 "Source" 部分**
   - Branch: `main`
   - Folder: `/ (root)`

5. **点击 "Save"**

6. **等待 1-5 分钟...**

7. **看到绿色消息** ✅
   ```
   Your site is published at:
   https://你的用户名.github.io/accounting-app
   ```

✅ **完成！** Pages 已启用

## 📱 在微信中使用

### 方法一：分享链接
1. 复制应用链接
2. 粘贴到微信聊天窗口
3. 点击链接打开

### 方法二：扫二维码
1. 打开 `qrcode-generator.html`
2. 输入应用链接
3. 生成二维码
4. 用微信扫描

### 方法三：添加到收藏
1. 在微信中打开应用
2. 右上角 ⋮ → "收藏"
3. 从"我的收藏"快速访问

## 🔄 更新应用

如果您修改了代码，想上传最新版本：

### 网页方式（最简单）
1. 进入仓库
2. 点击文件名
3. 点击编辑图标（铅笔）
4. 修改内容
5. 点击 "Commit changes"

### 自动脚本方式
1. 修改本地文件
2. 双击运行：
   - **Windows**: `upload-to-github.bat`
   - **Mac/Linux**: `upload-to-github.sh`
3. 按照提示输入信息
4. 自动上传！

## 🆘 遇到问题？

### 问题1：打开是空白页面
- ✅ 解决：等待 5-10 分钟，刷新浏览器

### 问题2：找不到应用链接
- ✅ 解决：检查 Pages 是否已启用（Settings → Pages）

### 问题3：上传文件失败
- ✅ 解决：检查网络连接，重新上传

### 问题4：在微信中打开是白页
- ✅ 解决：这是正常的加载延迟，等 10 秒再试
