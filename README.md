# 小宝记账

一款简单易用的记账Web应用，支持PC和移动端访问。

## 功能特性

- ✅ 快速记账（收入/支出）
- ✅ 分类管理（预设常用分类）
- ✅ 账单列表（按日期分组展示）
- ✅ 统计分析（按月/年/全部）
- ✅ 数据可视化（饼图展示）
- ✅ 数据导入导出（JSON格式）
- ✅ 本地存储（IndexedDB）
- ✅ 响应式设计（完美支持移动端）
- ✅ PWA支持（可添加到主屏幕）

## 技术栈

- React 18 + TypeScript
- Vite 5
- Tailwind CSS
- Zustand（状态管理）
- Recharts（数据可视化）
- IndexedDB（本地存储）
- React Router（路由管理）

## 开发指南

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

应用将在 http://localhost:3000 启动，支持热更新。

### 构建生产版本

```bash
npm run build
```

构建产物将输出到 `dist` 目录。

### 预览生产版本

```bash
npm run preview
```

## 使用说明

### 记账

1. 点击底部"记一笔"按钮
2. 选择收入或支出类型
3. 输入金额
4. 选择分类
5. 选择日期（默认今天）
6. 添加备注（可选）
7. 点击保存

### 查看明细

- 在"明细"页面查看所有记账记录
- 按日期分组显示
- 点击记录可查看详情和删除

### 查看统计

- 在"统计"页面查看收支统计
- 支持按月、按年、全部时间查看
- 饼图展示分类占比
- 列表显示详细金额和百分比

### 数据管理

在"设置"页面可以：
- 导出数据为JSON文件
- 从JSON文件导入数据
- 清空所有数据

## 移动端使用

### 添加到主屏幕

**iOS（Safari）：**
1. 打开网页
2. 点击底部分享按钮
3. 选择"添加到主屏幕"

**Android（Chrome）：**
1. 打开网页
2. 点击菜单
3. 选择"添加到主屏幕"

## 数据安全

- 所有数据保存在浏览器本地（IndexedDB）
- 不上传到任何服务器
- 建议定期导出数据备份
